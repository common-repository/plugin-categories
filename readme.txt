Categorize your WordPress Plugins just like you would posts. 

== Description ==

Plugin Categories gives you the abillity to categorize all your WordPress Plugins in the same manner you would categorize posts. 

It is a back end plugin that has no effect on anything in the front end of your website. It does not modify any plugin files for the categorization.

== Installation ==

1. Upload the folder `plugin-categories` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Start categorizing your plugins.

== Frequently Asked Questions ==

= How do I create categories? =

Find the "Categories"-menu under the "Plugins"-tab in the wordpress admin menu. Create categories just as you would for posts.

= How do I add a category to a plugin? =

Simply click the "Edit categories"-link below the plugin name in the Plugin-list and select the categories.
